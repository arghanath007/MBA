* XAT Exam Strategy. [IMPORTANT]
* GK -> No negative marking.
* GK -> 7-8marks -> Percentile doesn't matter -> Just get a Qualifying score/marks.
* DM -> 21Q, QADI -> 28Q, VARC -> 26Q => 75Q -> 170mins.
* No sectional boundation.
* Sectional Cut-off -> 8-9marks for each section. [IMPORTANT]

<img width="741" height="516" alt="image" src="https://github.com/user-attachments/assets/192341f8-b624-41c9-adad-095ae8facc79" />

* Give '50mins' to each section -> 150mins.  [IMPORTANT]
* Last '20mins' -> Strength/favourite section/questions. [IMPORTANT]
* Favourite section -> To maximize score.

* **QADI**. [IMPORTANT]
* For sectional cut-off -> 1 DI set, 5-6 quant questions.
* No individual cut-off.
* QA -> 17-18Qs.
* DI -> 11-10Qs.
* QADI -> 50-55mins => 10-11Q easily. [For avg. person]
* QADI -> 50-55mins => 16-18Q easily. [For person who like Quant]
* Trigonometry, Probability, Base system, Statistics -> 3-4Q come.
* Geometry -> 2-3Qs, Algebra ->4-5Qs, Arith ->4-5Qs, Number System ->1-2Qs, PnC -> 1Q.
